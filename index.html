Getting Started

You can usually just open index.html directly in your browser — no server required. Both Claude and Ollama requests go straight from your browser to their APIs.

1. Ollama — make sure it is running

The Ollama desktop app or ollama serve must be running so local models can be loaded.

2. Claude — enter your API key in Settings (⚙)

Your key is held in memory only and is deleted when you close the page. Requests are sent directly to Anthropic from the browser.

3. If your browser blocks a direct call (CORS)

Ollama in particular rejects requests from pages opened as a plain file, so this is common. The app automatically falls back to a local proxy server for either provider — but the fallback only works if that server is running. Start it in a terminal and open http://localhost:8080 instead of the file directly:

cd "/Users/geoffreyrockwell/Documents/AI Projects/spyral2LLM"
python3 server.py
